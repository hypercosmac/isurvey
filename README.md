## Isurvey Landing Page

A lightweight, static landing page for **Isurvey**, a design studio providing modern landing pages and marketing websites for other businesses.

### Running locally

- **Prerequisite**: Node.js installed.
- From the project root:

```bash
cd /Users/vikrammishra/Documents/GitHub/isurvey
npm install serve --no-save
npx serve .
```

Then open `http://localhost:3000` (or the URL that `serve` prints) in your browser.

### Customizing

- **Branding & copy**: Edit `index.html`.
- **Visual style** (tracing-paper grid, panels, typography): Edit `styles.css`.
- **Button behavior & smooth scrolling**: Edit `main.js`.

Update the external form URLs in `index.html` (search for `isurvey-intake-form`) to point to your live intake form.
